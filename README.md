# CGA-U-Net: Category Guided Attention Network for Brain Tumor Segmentation in MRI

This repo is the official implementation for [Category Guided Attention Network for Brain Tumor Segmentation in MRI](https://openreview.net/forum?id=YbsgVvEtZFw). The multimodal brain tumor dataset (BraTS 2019) could be acquired from [here](https://ipp.cbica.upenn.edu/).

## Requirements
- python 3.7
- pytorch 1.6.0
- torchvision 0.7.0
- pickle
- nibabel
